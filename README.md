# newsclip-digest-cli

Generate a simple markdown digest from a Newsclip category page.

A lightweight script designed for simple pipelines.

## Why this exists
- Reduce friction when prototyping pipelines and monitoring checks.
- Make it easy to validate inputs and produce stable output formats.

## More from Newsclip
- [Newsclip](https://newsclip.com/)
- [Newsclip Terms](https://newsclip.com/terms/)
- [Newsclip Business](https://newsclip.com/business/)

## Usage
Example:

```bash
npm install
npm run start -- https://newsclip.com/business/
```

## Notes
- This repo is intentionally small and readable.
- Best-effort parsing is used to keep dependencies minimal.
